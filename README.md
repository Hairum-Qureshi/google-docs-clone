### Core MVP

1. **Document ownership + access control**
   - One owner
   - Explicit invite list
   - Read/write permissions (even if binary)

2. **Presence system**
   - Show active users
   - Join/leave events
   - No polling

3. **Live collaborative editing**
   - Real-time text sync
   - No refresh
   - Deterministic order of updates

4. **Cursor + selection awareness**
   - Color-coded cursors
   - Cursor moves as document changes
   - Only for active users
